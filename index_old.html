About


Copionet is a free plagiarism detector tool initially created to help teachers to review the student's work. It supports the main doc file types: DOCX, ODT, TXT, PDF and much more!

This tool is developed by Jaume Mayor (IT Professor) after a long and arduous search without success for a convenient tool for quick student's assignments revision.
